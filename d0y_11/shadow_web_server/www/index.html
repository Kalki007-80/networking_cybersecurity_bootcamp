Welcome to Shadow Web Server

A hand-built HTTP server, serving this page straight from www/ — no frameworks involved.

This page is a static file

Everything you're looking at right now — this HTML, the stylesheet, and the script running below — is served by serve_static_file() from the /files/ route, exactly the way Apache or Nginx would serve a document root.
