About This Project

Shadow Web Server is a from-scratch HTTP/1.1 server built directly on top of Python's socket module — no Flask, no Django, no external dependencies.

  • Author: Mukesh S
  • Project: Day 11 Bootcamp Project
  • Stack: Python 3, sockets, threading
  • Version: 2.0

What it can do

Static file serving JSON REST API Threaded concurrency Access logging MIME type detection Config-driven setup
