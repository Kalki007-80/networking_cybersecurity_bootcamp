## Overview

This repository documents a 12-day intensive journey into Networking and Cybersecurity. The objective is to build strong networking fundamentals, develop practical Linux and Python skills, and complete a professional cybersecurity project.

Each day pairs theory with hands-on tooling. By the end of the roadmap, standard networking utilities such as `ping`, `traceroute`, `curl`, and `nmap` concepts have been studied and recreated in Python, culminating in a professional capstone project.

---

## Learning Objectives

- Learn computer networking from fundamentals
- Understand Linux networking
- Develop Python networking tools
- Learn packet analysis and network scanning
- Build a cybersecurity portfolio project
- Document the complete learning process

---

## Learning Roadmap

| Day | Topic | Project Deliverable |
|-----|-------|----------------------|
| Day 1 | Networking Fundamentals | Concepts & Notes — Network basics, devices, protocols, communication models |
| Day 2 | OSI Model & TCP/IP | Concepts & Notes — Layered networking, encapsulation, TCP/IP architecture |
| Day 3 | Linux Networking | Concepts & Notes — Linux networking commands, interfaces, routing and DNS |
| Day 4 | IP Addressing & Subnetting | Concepts & Notes — IPv4, CIDR, subnetting, network & host calculations |
| Day 5 | Routing & Switching | **Network Path Explorer** — DNS resolution, ICMP ping, traceroute, JSON/TXT reports |
| Day 6 | Wireshark | **Network Recon Scanner** — CIDR-based host discovery & port scanning |
| Day 7 | Nmap | Concepts & Notes — SSH, HTTPS, encryption, HTTP headers |
| Day 8 | Python Networking | **Service Whisperer** — TCP port scanner with banner grabbing and TLS support |
| Day 9 | Cybersecurity Fundamentals | **HTTP Investigator** — HTTP methods, custom headers, authentication and response timing |
| Day 10 | Packet Analysis | **TCP Client Explorer** — Raw socket HTTP client with manual request/response parsing |
| Day 11 | Security Dashboard | **Shadow Web Server** — Multi-threaded HTTP server with static files, REST APIs and logging |
| Day 12 | Final Project | **Shadow Network Operation Center (Shadow NOC)** — Unified reconnaissance platform combining DNS, ping, traceroute, port scanning, banner grabbing, TLS inspection, HTTP analysis, security headers, cookies and technology detection |

> **Note:** The topic names in the left column follow the official bootcamp curriculum. The project deliverables extend those topics into practical Python implementations. Each project folder contains its own `README.md` with setup instructions, features and usage examples.

## Ethical Use

Several projects in this repository actively probe network hosts through DNS lookups, ping sweeps, port scans, banner grabbing, TLS handshakes and HTTP requests.

These tools are intended **only for systems and networks you own or have explicit permission to test.**

Unauthorized scanning or reconnaissance against third-party systems may violate laws, regulations or acceptable-use policies.
