## Overview

This repository documents a 12-day intensive journey into Networking and Cybersecurity. The objective is to build strong networking fundamentals, develop practical Linux and Python skills, and complete a professional cybersecurity project.

Each day pairs theory with hands-on tooling — by the end of the roadmap, standard utilities like `ping`, `traceroute`, `nmap`, and `curl` have been studied *and* partially rebuilt from scratch in Python, culminating in a capstone project.

---

## Learning Objectives

- Learn computer networking from fundamentals
- Understand Linux networking
- Develop Python networking tools
- Learn packet analysis and network scanning
- Build a cybersecurity portfolio project
- Document the complete learning process

## Learning Roadmap

| Day | Topic | Project Deliverable |
|-----|-------|----------------------|
| Day 1 | Networking Fundamentals | Concepts & Notes — Network basics, devices, protocols, communication models |
| Day 2 | OSI Model & TCP/IP | Concepts & Notes — Layered networking, encapsulation, TCP/IP architecture |
| Day 3 | Linux Networking | Concepts & Notes — Linux networking commands, interfaces, routing, DNS |
| Day 4 | IP Addressing & Subnetting | Concepts & Notes — IPv4, CIDR, subnetting, network & host calculations |
| Day 5 | Routing & Switching | **Network Path Explorer** — DNS resolution, ICMP ping, traceroute, JSON/TXT reports |
| Day 6 | Wireshark | **Network Recon Scanner** — CIDR-based host discovery & port scanning |
| Day 7 | Nmap | Concepts & notes (SSH, HTTPS, encryption, HTTP headers) |
| Day 8 | Python Networking | **Service Whisperer** — TCP port scanning, banner grabbing, TLS support |
| Day 9 | Cybersecurity Fundamentals | **HTTP Investigator** — custom HTTP methods, headers, auth, timing analysis |
| Day 10 | Packet Analysis | **TCP Client Explorer** — HTTP over raw sockets, manual request/response parsing |
| Day 11 | Security Dashboard | **Shadow Web Server** — multi-threaded HTTP server, static files, REST APIs |
| Day 12 | Final Project | **Shadow Network Operation Center** — capstone recon tool unifying DNS, ping, traceroute, port scan, banners, TLS, HTTP, security headers, cookies, tech detection |

> **Note on this table:** the topic names in the left columns reflect the
> official bootcamp curriculum.
> Please read README, given in all days for exact scope.

##  Ethical Use

Several projects in this repository (Network Recon Scanner, Service Whisperer, Shadow NOC) actively probe network hosts — ping sweeps, port scans, banner grabs, TLS handshakes. **Only run these tools against systems and networks you own or have explicit permission to test.** Unauthorized scanning of third-party systems may violate computer-misuse laws and acceptable-use policies, independent of whether anything is exploited.
